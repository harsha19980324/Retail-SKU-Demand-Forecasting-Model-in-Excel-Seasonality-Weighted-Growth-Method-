## 🚀 Overview

This project presents a complete **Excel-based demand forecasting solution** for 25 retail SKUs across multiple categories (Men, Women, Kids, Unisex, Sports).

The model uses **historical sales data (2022–2025)** to generate a **monthly forecast for 2026**, incorporating:

- Trend analysis  
- Seasonality patterns  
- Weighted growth logic

## ⚙️ Forecast Methodology

### 🔹 Weighted Moving Average (WMA)

- 50% → Most recent year  
- 30% → Middle year  
- 20% → Oldest year  

---

### 🔹 Growth Cap Logic

- Minimum: **-15%**  
- Maximum: **+20%**  

👉 Prevents unrealistic spikes or drops  

---

### 🔹 Seasonality Adjustment

- Monthly demand scaled using historical seasonal index  

---

### 🔹 Monthly Forecast Formula

**Annual Forecast × (Seasonality Index ÷ 12)**
